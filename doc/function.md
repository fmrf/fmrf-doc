# 胖蚊子兔场 (软件) 功能列表

主要功能: (暂定)

```
+ 数据采集与监视
  + 兔场监视 (monitor)
    + 传感器
      - 兔笼温湿度
      - 兔笼重量
      - 兔笼供水监测
      - 场地 O2, CO2 浓度
      - 风速风向
    + 视频音频 (Audio Video, AV)
      - 兔笼麦克风
      - 兔笼摄像头
      - 场地麦克风
      - 场地摄像头
    + 合成监测
      - 噪声监测
      - 饲料用量监测
      - 兔子状态 (活跃度, 健康) 监测
  + 自监视 (Self Monitor, SM)
    - 操作日志
    - 服务器状态
    - 网络状态
    - 兔场设备状态
    - 电源状态
    - 高可用状态
    - 日志查看
  + 异常警告
    - 警告规则
    - 警告生成与查看
    - 警告推送
    - 手机端消息推送

+ 设备遥控
  - 兔笼锁
  - 兔笼供水
  - 加料机
  - 兔笼扬声器 (放音乐)
  - 通风设备
  - 温湿度控制设备
  - 照明设备
  - (胖蚊子黑蜘蛛)

+ 自动控制
  - 规则设定
  - 运行日志

+ 兔场管理
  - 兔场基本信息
  - 兔场区域管理
  - 兔笼管理
  - 兔场设备管理
  - 兔只管理
  - 耗材管理 (饲料管理)
  - 历史数据统计分析
  - 数据导出

+ 系统管理
  - 账号管理
  - 角色管理
  - 系统设置
  - 数据清理
  - 数据备份

```

其它功能:

